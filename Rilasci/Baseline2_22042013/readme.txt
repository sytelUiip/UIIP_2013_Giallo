GRUPPO GIALLO - Baseline2_22042013

In questa fase sono state rilasciate le seguenti cartelle:
- Documentazione: contenente il documento di architettura, il manuale di installazione e il manuale utente.
- file_conf_tomcat: contenente i file context.xml e server.xml per la configurazione di Tomcat.
- keystore: certificato per il protocollo HTTPS da copiare in C:/ .
- SQLScripts: contenente gli script SQL aggiornati.
- WAR: contenente i .war dell'applicazione realizzata in questa release.
- Web_Server: contenente il web server Apache Tomcat.
- workspace_gruppo_giallo: contenente l'archivio della workspace del progetto sviluppato.
- WSDL: contenente le interfacce tra i sistemi (WSDL dei WS) per tutti i processi previsti.
